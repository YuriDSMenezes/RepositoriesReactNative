## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/lukemorales/react-rocketshoes

# Go into the repository
$ cd react-rocketshoes

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
